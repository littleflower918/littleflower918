#### 对象的取值

共有两种方法

1. 打点调用
2. 方括号调用



```javascript
 var container = {
        a : "111",
        b : "222",
        c : "333",
        "favorite-idol" :["段星星","孙滢皓"]
    };
//打点调用
   console.log(container.a);
//方括号调用
   console.log(container["favorite-idol"]);
```
